# POLÍTICA DE PRIVACIDAD

**Última actualización:** 12 de mayo de 2026

Esta aplicación utiliza la API de Meta (Facebook e Instagram) para automatizar la creación y publicación de contenido en redes sociales.

## 1. Responsable del tratamiento

El responsable del tratamiento de los datos es el desarrollador de la aplicación.
Contacto: soporte del desarrollador.

## 2. Información que se recopila

La aplicación puede acceder y procesar únicamente la información estrictamente necesaria para su funcionamiento:

- Identificadores de cuenta de Instagram y Facebook (ID de usuario o página)
- Nombre de usuario público
- Permisos otorgados por el usuario para publicación de contenido
- Tokens de acceso generados por la plataforma Meta (OAuth)

La aplicación no solicita ni almacena datos personales sensibles.

## 3. Finalidad del uso de datos

Los datos recopilados se utilizan exclusivamente para:

- Publicación automática de contenido en cuentas autorizadas
- Programación de publicaciones
- Gestión de sesiones de autenticación con la API de Meta
- Operación del sistema automatizado de generación de contenido

No se utiliza la información para fines distintos a los descritos.

## 4. Almacenamiento y seguridad

- Los tokens de acceso pueden almacenarse de forma local y/o en el entorno del usuario.
- Se aplican medidas razonables de seguridad para evitar accesos no autorizados.
- La aplicación no realiza transferencia de datos a servidores de terceros no autorizados.

## 5. Compartición de información

La información NO es vendida, alquilada ni compartida con terceros.

Únicamente se comparte información con los servicios necesarios para la operación:

- Meta Platforms, Inc. (Facebook e Instagram API)

## 6. Retención de datos

Los datos se conservan únicamente mientras la integración con la API de Meta esté activa.

El usuario puede revocar el acceso en cualquier momento.

## 7. Eliminación de datos

El usuario puede solicitar la eliminación de datos mediante:

- Revocación de permisos desde su cuenta de Facebook/Instagram
- Eliminación de la conexión dentro de la aplicación

Una vez revocados los permisos, el acceso a tokens y datos asociados deja de ser válido de inmediato.

## 8. Derechos del usuario

El usuario puede en cualquier momento:

- Acceder a sus datos procesados
- Solicitar la eliminación de sus datos
- Revocar permisos de acceso
- Detener el uso de la integración con Meta

## 9. Cumplimiento de plataformas

Esta aplicación cumple con las políticas de la API de Meta Platforms, Inc. y sus requisitos de uso de datos.

## 10. Servicios de terceros

Esta aplicación utiliza exclusivamente:

- Meta Platforms, Inc. (Facebook e Instagram API)

No se integran servicios adicionales de terceros no declarados.

## 11. Seguridad de autenticación

Los tokens OAuth se manejan como credenciales sensibles y no son expuestos públicamente. El acceso está limitado al funcionamiento interno de la aplicación.

## Eliminación de datos de usuario

El usuario puede solicitar la eliminación total de sus datos asociados a la aplicación en cualquier momento.

Los datos que pueden ser eliminados incluyen:

- Tokens de acceso de Meta (Facebook/Instagram)
- Identificadores de cuenta asociados a la integración
- Permisos otorgados a la aplicación

### Métodos de eliminación disponibles:

1. **Revocación directa desde Meta:**
   El usuario puede eliminar el acceso desde la configuración de su cuenta de Facebook o Instagram, en la sección de aplicaciones conectadas.

2. **Eliminación dentro del sistema:**
   El usuario puede solicitar la eliminación de datos enviando una solicitud al soporte del desarrollador.

### Efecto de la eliminación:

- El token de acceso deja de ser válido de forma inmediata.
- Se interrumpe cualquier automatización activa.
- No se conserva copia de los datos eliminados en el sistema.

### Tiempo de procesamiento:

La eliminación se ejecuta de forma inmediata tras la revocación o solicitud del usuario.

Correo: soporte del desarrollador de la aplicación
